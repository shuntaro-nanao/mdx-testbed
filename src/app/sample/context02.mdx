# Context 02

This is a sample MDX file that demonstrates basic MDX features.

## Features

- Markdown support
- JSX components
- Frontmatter metadata

## Example Component

You can use React components in MDX:

```jsx
<button onClick={() => alert('Hello!')}>
  Click me
</button>
```

## Code Blocks

You can also include code blocks:

```javascript
const greeting = 'Hello, MDX!';
console.log(greeting);
```

## Conclusion

MDX combines the best of Markdown and React components.
