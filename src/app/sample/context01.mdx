# Sample MDX Page

This is a sample page written in MDX.

## Features

- Markdown syntax works as expected
- You can use React components
- Code blocks are supported

```js
const greeting = "Hello, MDX!";
console.log(greeting);
```

## React Component Example

You can use React components directly in your MDX:

<div style={{ padding: '1rem', backgroundColor: '#f0f0f0', borderRadius: '4px' }}>
  This is a styled div using React inline styles
</div>
